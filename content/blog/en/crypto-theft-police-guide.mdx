## Why Report to Police?

Many cryptocurrency theft victims hesitate: **Is reporting useful?** The answer is: **Yes, and it's crucial**.

### Three Core Values of Reporting

1. **Legal Procedure Initiation**
   - Only after reporting can police formally intervene in investigation
   - Only after case filing can legal documents be issued requiring exchange cooperation
   - Only judicial procedures can freeze assets involved

2. **Possibility of Joint Cases**
   - Your case may be related to multiple cases nationwide
   - Single case amounts may be small, but combined may reach major case standards
   - Helps police identify criminal gangs

3. **Evidence Preservation**
   - Police intervention can preserve key evidence
   - Prevents evidence destruction or tampering
   - Lays foundation for subsequent civil litigation

## Preparation Before Reporting

Adequate preparation can greatly improve filing success rates. Here's a detailed preparation checklist:

### Basic Identity Materials

| Material | Purpose | Notes |
|----------|---------|-------|
| ID card original and copy | Prove reporter identity | Essential |
| Proof of residence | Determine jurisdiction | If reporting outside household registration |
| Contact information | Police follow-up contact | Keep open |

### Core Evidence Materials

#### 1. Transaction Record Proof

**Blockchain explorer screenshots/exports**:
```
Required information:
✓ Transaction hash (Tx Hash)
✓ Transaction time (precise to seconds)
✓ Sender address (your address)
✓ Receiver address (attacker address)
✓ Transfer amount
✓ Transaction status (Success)
```

**Recommended explorers**:
- ETH chain: Etherscan.io
- BSC chain: BscScan.com
- TRON chain: Tronscan.org
- Polygon: Polygonscan.com

#### 2. Wallet Ownership Proof

Evidence proving the stolen wallet belongs to you:

| Proof Method | Applicable Scenario | Effectiveness |
|--------------|--------------------|---------------|
| Creation records | Exchange withdrawal records | ⭐⭐⭐⭐⭐ |
| Historical transactions | Original deposit records to this address | ⭐⭐⭐⭐⭐ |
| Seed phrase/private key | Export and seal for submission | ⭐⭐⭐⭐ |
| Wallet screenshot | App interface showing address | ⭐⭐⭐ |

#### 3. Theft Process Description

Write a detailed "Police Report" or "Situation Explanation":

```
Police Report Basic Structure:

I. Reporter Information
   - Name, ID number, contact information

II. Case Process
   - When discovered stolen
   - Operations before theft (what websites visited, what links clicked)
   - Process of discovering theft
   - Stolen amount and currency

III. Fund Flow
   - Fund sender address
   - Attacker address
   - Known fund destination

IV. Related Evidence
   - Transaction hash list
   - Chat records (if any)
   - Website screenshots (if any)

V. Request
   - Request case filing and investigation
   - Recover stolen assets
   - Combat crime
```

#### 4. Value Proof

Prove the value of stolen assets:

- **Purchase records**: Exchange order records for buying USDT
- **Withdrawal records**: Records of withdrawing from exchange to stolen wallet
- **Price proof**: Token price screenshot on day of incident (CoinMarketCap, etc.)
- **Exchange rate conversion**: Convert to RMB amount using exchange rate on day of incident

### Technical Evidence Materials (If Available)

| Fund flow diagram | Drawn using tracking tools | Visual presentation |

## Where to Report?

Choosing the right reporting location can improve filing efficiency.

### Reporting Location Priority

```
Priority 1: Public security organs in your location (cybersecurity department)
├─ Advantage: Convenient for you to cooperate with investigation
├─ Applicable: Most situations
└─ Note: Choose district-level or higher cybersecurity brigade

Priority 2: Public security organs where attacker is located
├─ Advantage: Convenient for arresting suspects
├─ Applicable: Attacker location identified
└─ Note: Requires clear location information

Priority 3: Public security organs where funds flowed
├─ Advantage: Convenient for obtaining exchange data
├─ Applicable: Funds entered domestic exchanges
└─ Note: Requires exchange cooperation

Priority 4: Public security organs where main server is located
├─ Advantage: Convenient for evidence collection
├─ Applicable: Involving phishing websites/fake apps
└─ Note: Requires server location information
```

### Specific Reporting Channels

#### Channel 1: Local Public Security Bureau Cybersecurity Department (Recommended)

**Applicable scenarios**: Most cases

**Process**:
1. Bring materials to local public security bureau
2. Request transfer to cybersecurity department (Cybercrime Investigation Brigade)
3. Submit report materials
4. Obtain "Case Acceptance Receipt"

**Notes**:
- Go on weekday mornings if possible
- Make appointments in advance (supported in some cities)
- Prepare multiple copies of materials

#### Channel 2: Cybercrime Reporting Website

**Website**: https://www.cyberpolice.cn (China)

**Applicable scenarios**: Initial reports, clue submission

**Process**:
1. Register account and verify identity
2. Fill out report form
3. Upload evidence materials
4. Save report number

**Notes**:
- Online reporting is not equivalent to formal reporting
- Still need offline material submission
- Suitable as supplementary channel

#### Channel 3: National Anti-Fraud Center App

**Applicable scenarios**: Telecom fraud cases

**Process**:
1. Download and register National Anti-Fraud Center app
2. Select "I want to report"
3. Fill in case information
4. Upload evidence

#### Channel 4: Emergency Police Hotline

**Applicable scenarios**: Emergency situations, ongoing crimes

**Process**:
1. Dial emergency number (110 in China, 911 in US)
2. Explain basic case situation
3. Follow instructions to go to police station or wait for police

## At the Police Station: How to Communicate with Police

Communication methods during reporting directly affect filing results.

### Reporting Script Guide

#### Opening (30 seconds to capture key points)

```
Recommended script:

"Officer, I'd like to report a case. My cryptocurrency was stolen,
involving approximately XX million RMB. I've prepared complete evidence materials,
including transaction records, fund flows, and wallet ownership proof."
```

#### Key Information Emphasis

| Point | Explanation | Script Example |
|-------|------------|----------------|
| Clear amount | Convert to RMB | "Involving approximately 500,000 RMB" |
| Complete evidence | Emphasize preparation | "I've prepared all evidence materials" |
| Technical clarity | Explain in plain language | "Like my digital wallet was hacked" |
| Cooperative attitude | Show willingness to cooperate | "I'm available anytime to cooperate with investigation" |

#### Common Police Questions and Responses

**Q1: "This is investment loss, not a case"**

**Response**:
- Clearly distinguish: "This isn't investment, it's hacking/fraud"
- Provide evidence: Show authorization records or phishing website screenshots
- Emphasize illegality: "The other party obtained my assets through deceptive means"

**Q2: "Cryptocurrency isn't legally protected"**

**Response**:
- Cite regulations: "According to 'Notice on Preventing Bitcoin Risks', Bitcoin and others are virtual commodities protected by law"
- Emphasize property rights: "My virtual property was stolen, Criminal Law Article 264 should apply"
- Provide cases: Filing cases from other regions

**Q3: "Technically impossible to trace"**

**Response**:
- Provide analysis reports: Professional on-chain tracking reports
- Explain feasibility: "Funds entered XX exchange, KYC information can be obtained"
- Provide assistance: "We have professional technical teams to assist investigation"

### Key Factors for Successful Filing

Based on our experience, these factors significantly improve filing rates:

| Factor | Impact | Suggestion |
|--------|--------|-----------|
| Amount size | Larger amounts easier to file | Clarify involved amount |

## Process After Reporting

### Normal Process

```
Report → Acceptance → Preliminary investigation → Filing → Investigation → Closure

Time nodes:
- Acceptance: Issue "Case Acceptance Receipt" on the spot
- Preliminary investigation: Usually completed within 30 days
- Filing: Decide whether to file after preliminary investigation
- Investigation: Begin after filing
```

### Key Documents

| Document | Purpose | Acquisition Method |
|----------|---------|-------------------|
| Case acceptance receipt | Proof of acceptance | Obtained on the spot when reporting |
| Filing decision | Formal filing proof | Issued after filing |
| Non-filing notice | Explanation for non-filing | Can apply for review |
| Investigation letter | Require third-party cooperation | Apply after filing |

### If Police Don't File

**Step 1: Understand reasons**
- Request "Non-filing Notice"
- Ask for detailed reasons for non-filing

**Step 2: Apply for review**
- Apply for review to original public security organ (within 7 days)
- Apply for re-review to higher-level public security organ (within 30 days)

**Step 3: Apply to procuratorate for filing supervision**
- Submit application to same-level procuratorate
- Procuratorate has authority to require public security to explain non-filing reasons

**Step 4: Criminal private prosecution**
- If clear evidence is available, can directly file private prosecution with court

## Tips for Cooperating with Police

### DOs and DON'Ts for Investigation Cooperation

#### ✅ DO

- [ ] Keep phone open, respond promptly to police contact
- [ ] Supplement materials as required by police
- [ ] State truthfully, don't exaggerate or conceal
- [ ] Provide technical assistance (if needed)
- [ ] Regularly inquire about case progress (1-2 times per month is appropriate)

#### ❌ DON'T

- [ ] Frequently urge (multiple times per week)
- [ ] Request police to reveal case details
- [ ] Contact suspects or witnesses on your own
- [ ] Post case details on social media
- [ ] Pressure or threaten police

### Following Up on Case Progress

**Reasonable follow-up frequency**:
- 1 month after filing: First inquiry
- Every 1-2 months thereafter
- Respond promptly for major developments

**Inquiry methods**:
- Phone inquiry to handling officer
- In-person inquiry at police station
- Formal inquiry through lawyer

## Special Scenario Handling

### Scenario 1: Overseas Theft

**Characteristics**: Attacker is overseas, funds may flow overseas

**Response strategies**:
1. Still report domestically (for cases where funds flow into domestic territory)
2. Simultaneously report to police in attacker's country (if information is clear)
3. Coordinate through Interpol
4. Monitor whether funds flow into domestic exchanges

### Scenario 2: Multiple People Defrauded (Series Cases)

**Characteristics**: Same gang committed crime, multiple victims

**Response strategies**:
1. Proactively contact other victims
2. Establish contact group
3. Elect representatives for unified reporting
4. Emphasize "series case" nature

**Advantages**:
- Large cumulative amount, easier to file
- Evidence corroborates each other
- High police attention level

### Scenario 3: Involving Exchanges

**Characteristics**: Funds entered or passed through exchanges

**Response strategies**:
1. Clearly identify exchange in report materials
2. Provide exchange address evidence
3. Apply for police to send letter requiring exchange cooperation
4. Lawyer letter if necessary

## FAQ

### Q1: How long after reporting will the case be filed?

**A**: Legal provisions stipulate a 30-day preliminary investigation period, extendable for complex cases. General process:
- Within 7 days: Decide whether to accept
- Within 30 days: Complete preliminary investigation
- After preliminary investigation: Decide whether to file

### Q2: What are the filing standards for cryptocurrency cases?

**A**: Standards vary by region, generally refer to:
- Theft: Above 1,000-3,000 RMB (varies by region)
- Fraud: Above 3,000-10,000 RMB
- Large amount: Above 30,000-100,000 RMB
- Particularly large amount: Above 300,000-500,000 RMB

### Q3: What if police say "impossible to trace"?

**A**: You can:
1. Provide professional on-chain analysis reports
2. Explain fund flows (especially entering exchanges)
3. Seek assistance from professional agencies
4. Report to higher authorities or procuratorate

### Q4: Will reporting affect my other cryptocurrency assets?

**A**: No. Reporting targets stolen assets and won't affect your other legitimate assets.

### Q5: Can I report to multiple places simultaneously?

**A**: Yes, but recommended:
- Priority to report in your location
- If you know attacker location, can simultaneously report there
- Inform police you've reported elsewhere
- Avoid duplicate filings
