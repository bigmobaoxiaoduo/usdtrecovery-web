## Common Fake Wallet Scam Tactics

### Tactic 1: High-Fake Official Websites

Scammers create websites almost identical to official ones:
- Subtle domain differences (imtoken.im vs imtoken.com)
- Interface completely copied
- Even provide "customer service" functions

### Tactic 2: App Store Phishing

- List fake wallets on third-party app stores
- Use similar icons and names
- Fake positive reviews to create false credibility

### Tactic 3: Community Promotion

- Share "benefits" in WeChat groups, Telegram groups
- Claim "airdrops" or "mining" require downloading specific wallets
- Induce users to import private keys through social engineering

## How to Identify Fake Wallets

1. **Check official website domain**
   - Carefully check every letter
   - Use HTTPS
   - Check SSL certificate

2. **Download only from official channels**
   - Official website download links
   - Official app store accounts
   - Verify developer information

3. **Never import private keys into any webpage**
   - Real wallets won't ask you to enter seed phrases on web pages
   - Anyone asking for seed phrases is a scammer

## What to Do If Stolen

If you unfortunately downloaded a fake wallet and imported assets:
1. Immediately transfer remaining assets to a newly created secure wallet
2. Save fake wallet app and website screenshots
3. Record transaction hashes and scammer addresses
4. Contact professional team to assess recovery possibility

Prevention is always better than remedy. Please stay vigilant!

## Additional: Wallet Security Best Practices

### Download Verification Checklist

- [ ] Verify official website domain (check for HTTPS)
- [ ] Download from official sources only
- [ ] Check app developer information
- [ ] Verify download counts and ratings
- [ ] Read user reviews carefully
- [ ] Check app update history

### Red Flags of Fake Wallet Apps

| Red Flag | Explanation |
|----------|-------------|
| Requests seed phrase during setup | Real wallets generate seed phrases locally |
| Requires internet to generate wallet | Wallets should work offline for creation |
| Asks for personal information | Decentralized wallets don't need KYC |
| Poor grammar/spelling | Professional wallets have quality standards |
| No open source code | Legitimate wallets often have open source components |
