## 钱包安全的基础认知

### 什么是加密货币钱包？

加密货币钱包不是"存储"币的地方，而是**管理私钥的工具**。

- **私钥**：控制资产的唯一凭证
- **公钥/地址**：接收资产的"账号"
- **助记词**：私钥的人类可读形式

**核心原则**：
> 谁掌握私钥，谁就拥有资产。钱包服务商不保存你的资产，只提供管理工具。

### 安全事件的三大根本原因

| 排名 | 原因 | 占比 | 典型场景 |
|------|------|------|----------|
| 1 | 助记词泄露 | 35% | 截图存手机、发给"客服"、存云盘 |
| 2 | 恶意合约授权 | 28% | 点击钓鱼链接、授权恶意合约 |
| 3 | 假钱包APP | 20% | 下载山寨钱包、导入助记词 |
| 4 | 设备被入侵 | 12% | 手机/电脑中木马 |
| 5 | 其他 | 5% | 社工欺骗、暴力破解等 |

### 两种钱包的核心区别

| 特性 | 冷钱包（硬件钱包） | 热钱包（软件钱包） |
|------|------------------|------------------|
| **私钥存储** | 离线设备 | 联网设备 |
| **安全性** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| **便捷性** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

| **价格** | $50-300 | 免费 |

### 选择建议

#### 使用冷钱包的场景

✅ **强烈推荐使用冷钱包**：
- 持有价值 > $10,000 的加密资产
- 长期持有不频繁交易
- 企业或机构资产管理
- 对安全性要求极高

#### 使用热钱包的场景

✅ **可以使用热钱包**：

**热钱包使用原则**：
- 只存放"能承受损失"的金额

- 不存储助记词在手机或电脑

## 助记词安全：你的数字生命线

### 助记词是什么？

助记词（Recovery Phrase / Seed Phrase）是**生成私钥的种子**，通常由12或24个英文单词组成。

**关键认知**：
- 任何人拥有助记词 = 拥有钱包的完全控制权
- 助记词无法重置或找回
- 助记词泄露 = 资产必然被盗

### 助记词存储的DO和DON'T

#### ❌ 绝对禁止的做法

| 危险行为 | 风险等级 | 真实案例 |
|----------|----------|----------|
| 截图保存在手机相册 | 🔴 极高 | 手机被黑，损失30 ETH |
| 存储在云服务 | 🔴 极高 | iCloud被盗，损失10万USDT |
| 通过社交软件发送 | 🔴 极高 | 发给"客服"，瞬间被盗 |
| 保存在电脑文档 | 🟠 高 | 电脑中木马，助记词被盗 |
| 拍照存手机 | 🟠 高 | 手机丢失，资产被盗 |
| 告诉任何人 | 🔴 极高 | 熟人作案，无法追回 |

#### ✅ 推荐的安全做法

**方式一：纸质备份（基础方案）**

```
操作步骤：
1. 在纸上手抄助记词（至少2份）
2. 核对3遍确保无误
3. 存放在防火防水的安全地点
4. 可考虑银行保险箱
```

**方式二：金属助记词板（推荐方案）**

- 防火（可达1000°C+）
- 防水防腐蚀
- 防物理损坏
- 适合长期存储

**推荐产品**：
- Cryptosteel Capsule ($80)
- Billfodl ($40)
- 国产助记词钢板 ($15-30)

**方式三：分层存储（高级方案）**

将24个单词分成多组，分别存储：

```
示例（3选2方案）：
- 地点A：单词 1-8
- 地点B：单词 9-16  
- 地点C：单词 17-24

安全性：
- 任何两个地点可以恢复钱包
- 单个地点泄露不会导致资产损失
- 即使一个地点被毁，仍可恢复
```

### 助记词验证清单

备份助记词后，务必完成以下检查：

- [ ] 手抄字迹清晰可辨
- [ ] 已核对3遍，确认无误
- [ ] 已按顺序编号（1-12或1-24）
- [ ] 已测试恢复钱包（使用备用设备）
- [ ] 存放在安全地点
- [ ] 家人或信任的人知道存放位置（可选）

## 热钱包安全配置指南

### MetaMask 安全配置

#### 基础设置

```
1. 设置强密码
   - 至少12位
   - 包含大小写字母、数字、符号
   - 不使用常见词汇

2. 启用生物识别（移动端）
   - 指纹或面部识别
   - 增加一层保护

3. 自动锁定设置
   - 设置1-5分钟自动锁定
   - 防止离开后被盗用
```

#### 高级安全设置

**1. 隐藏代币金额**
```
设置路径：
Settings → Security & Privacy → Show balance and token price ON/OFF
```
- 防止窥屏泄露资产信息

**2. 禁用ENS自动解析**
```
Settings → Privacy → ENS domain lookup → OFF
```
- 防止访问恶意ENS域名

**3. 关闭不必要的API**
```
Settings → Security & Privacy →
- Phishing detection: ON
- Participate in MetaMetrics: OFF（可选）
```

### Trust Wallet 安全配置

```
推荐设置：
✓ 启用应用锁（PIN或生物识别）
✓ 启用交易确认（需要密码/指纹）
✓ 禁用DApp浏览器（如不需要）
✓ 定期检查已授权DApp
```

## 授权管理：最容易被忽视的风险

### 什么是代币授权？

当你使用DeFi协议（如Uniswap、Aave）时，需要"授权"合约访问你的代币。这个授权：
- 通常是无上限的（允许转移全部余额）
- 长期有效（除非主动撤销）
- 被恶意合约利用会导致资产被盗

### 真实案例：授权被盗

**案例**：用户参与某"空投"活动，授权了恶意合约，1小时后50万USDT被转走。

**原因**：恶意合约利用了无上限授权，在获得授权后可以随时转移用户资产。

### 授权管理最佳实践

#### 1. 定期检查授权

**推荐工具**：
- **Revoke.cash** (https://revoke.cash) - 最流行
- **Unrekt** (https://app.unrekt.net) - 多链支持
- **DeBank** (https://debank.com) - 全面的资产管理

**检查频率**：
- 重度DeFi用户：每周一次
- 普通用户：每月一次
- 发现异常后立即检查

#### 2. 使用限额授权

部分钱包支持"限额授权"：
```
MetaMask操作：
1. 授权时点击"Edit permission"
2. 选择"Custom spend limit"
3. 设置具体金额（如1000 USDT）
4. 确认授权
```

#### 3. 及时撤销无用授权

```
使用Revoke.cash撤销步骤：
1. 访问 https://revoke.cash
2. 连接钱包（使用只读模式更安全）
3. 查看所有授权列表
4. 对不再使用的授权点击"Revoke"
5. 在钱包中确认交易（需支付少量Gas费）
```

### 授权安全清单

- [ ] 每月检查一次所有授权
- [ ] 撤销不再使用的授权
- [ ] 对新项目使用限额授权
- [ ] 不授权来路不明的合约
- [ ] 定期检查授权列表

## 防范假钱包和钓鱼攻击

### 假钱包APP识别

#### 常见套路

| 套路 | 特征 | 防范方法 |
|------|------|----------|
| 高仿官网 | 域名细微差别 | 仔细核对域名，使用书签 |
| 山寨APP | 应用商店盗版 | 只从官网下载 |
| 假客服 | 主动私信提供"帮助" | 官方从不主动私信 |
| 空投诱导 | 下载APP领取空投 | 警惕天上掉馅饼 |

#### 安全下载检查清单

下载钱包APP前检查：

- [ ] 官网域名正确（使用https）
- [ ] 从官网提供的链接下载
- [ ] 核对应用开发者信息
- [ ] 查看下载量和评分
- [ ] 阅读用户评论
- [ ] 校验应用签名（高级）

### 钓鱼攻击防范

#### 常见钓鱼手法

**1. 假网站钓鱼**
```
真实域名：metamask.io
钓鱼域名：metamask.io-wallet.com
           metamask-wallet.io
           metamask-io.com
```

**2. 假客服钓鱼**
- 在社交媒体上伪装成官方客服
- 声称可以"解决你的问题"
- 诱导你提供助记词或私钥

**3. 假空投钓鱼**
- 声称有高额空投
- 诱导访问钓鱼网站连接钱包
- 诱导授权恶意合约

#### 防钓鱼实用技巧

```
1. 使用官方书签
   - 将常用网站添加到书签
   - 只通过书签访问

2. 仔细核对域名
   - 检查每一个字符
   - 注意Unicode字符欺骗（如ｍetamask.io）

3. 不轻易点击链接
   - 即使是朋友发送的
   - 即使是邮件中的"官方"链接

4. 使用硬件钱包确认
   - 所有交易在硬件设备上确认
   - 屏幕上显示真实的收款地址
```

## 设备与环境安全

### 手机安全

```
必做：
✓ 设置强密码或生物识别
✓ 开启"查找我的设备"
✓ 保持系统更新
✓ 仅从官方应用商店下载APP
✓ 安装安全软件

禁做：
✗ 越狱/Root
✗ 安装来路不明的APP
✗ 点击可疑短信链接
✗ 连接不安全的WiFi
```

### 电脑安全

```
必做：
✓ 使用正版操作系统
✓ 安装并更新杀毒软件
✓ 使用硬件钱包时，电脑保持干净
✓ 定期清理浏览器插件
✓ 使用密码管理器

禁做：
✗ 在公共电脑登录钱包
✗ 下载破解软件
✗ 打开可疑邮件附件
✗ 使用来路不明的浏览器插件
```

### 网络安全

```
安全做法：
✓ 使用VPN保护隐私
✓ 避免公共WiFi进行敏感操作
✓ 使用移动数据或可信网络
✓ 启用路由器的防火墙
```

## 应急响应：如果怀疑被盗

### 立即行动清单

发现可疑情况时，**立即执行**：

```
第1步：断开网络连接
- 关闭WiFi和移动数据
- 防止进一步攻击

第2步：转移剩余资产
- 使用干净的设备创建新钱包
- 立即转移所有剩余资产
- 使用不同的助记词

第3步：检查授权
- 使用revoke.cash撤销所有授权
- 防止进一步损失

第4步：保存证据
- 截图所有相关信息
- 记录交易哈希
- 保存攻击者地址

第5步：寻求专业帮助
- 联系专业安全团队
- 评估是否需要报警
```
